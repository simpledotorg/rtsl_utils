

# Prerequisite

brew install openjdk maven


# To compile the whole project

```
cd rtsl_util
mvn clean install
```



# To test the cucumber POC

```
cd rtsl_util/CommonUtils/Dhis2CucumberTestTool
bash ./src/test/resources/local_tests/tests.ad.local.sh 
```
